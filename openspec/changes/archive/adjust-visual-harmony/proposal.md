# 变更提案：视觉风格调优（冰雪毛玻璃）

## 概述
解决当前"黑色终端风"与"明亮雾凇背景"视觉冲突严重的问题。采用"冰雪毛玻璃 (Frosted Glass)"风格，使界面与背景完美融合。

## 需求场景
### 场景 1：视觉和谐
- Given 访客浏览博客
- When 页面加载
- Then 界面组件（导航栏、卡片）应呈现半透明白色（毛玻璃效果），而非突兀的黑块。
- And 文字颜色应清晰易读（深灰/黑），强调色调整为冷蓝色，呼应背景中的天空与水面。

## 技术方案
1. **回退主题变体**：
   - 将 `_config.icarus.yml` 中的 `variant` 改回 `default`（亮色基础）。
2. **重写 CSS (`source/css/custom.css`)**：
   - **背景**：`rgba(255, 255, 255, 0.6)` + `backdrop-filter: blur(12px)`。
   - **阴影**：去除厚重阴影，使用轻柔光晕。
   - **去色**：移除之前的绿色终端边框和文字强制样式。
   - **字体**：恢复主题默认的易读字体颜色。

## 影响范围
- 全站视觉风格（由暗转亮）。
