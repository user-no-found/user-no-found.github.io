# 变更提案：现代简约科技感UI重设计

## 概述
全面重设计博客UI，替换原有全息投影风格，采用现代简约科技感设计。

## 设计原则
- **简约**：去除多余动画和装饰
- **现代**：大圆角、毛玻璃、微交互
- **科技感**：深色蒙版 + 青蓝强调色 + 细线条

## 设计规范

### 配色方案
- 背景：保留雾凇.jpg + rgba(0,0,0,0.6) 深色蒙版
- 强调色：#00D9FF（青蓝）
- 卡片背景：rgba(255,255,255,0.08)
- 文字：#E5E5E5（主）、#A3A3A3（次）
- 边框：rgba(255,255,255,0.1)

### 卡片样式
- 圆角：16px
- 毛玻璃：backdrop-filter: blur(20px)
- 边框：1px solid rgba(255,255,255,0.1)
- 悬停：微抬升 + 边框亮度增加
- 无持续动画

### 导航栏
- 固定顶部
- 毛玻璃背景
- 底部细线边框

### 动效
- 仅悬停过渡（transform, box-shadow）
- 过渡时长：0.3s ease
- 无旋转、无扫描线、无持续动画

## 影响范围
- 重写 `/source/css/custom.css`
- 可能调整 `_config.icarus.yml` 中的 variant
