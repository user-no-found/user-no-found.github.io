# 变更提案：全息投影风格UI

## 概述
为博客模块添加全息投影科技感风格，保留原有背景图片（雾凇.jpg）。

## 需求场景
### 场景 1：卡片展示
- Given 用户打开页面
- When 看到卡片/widget模块
- Then 看到彩虹渐变发光边框，有全息投影感

### 场景 2：悬停交互
- Given 用户浏览页面
- When 鼠标悬停在卡片上
- Then 边框流光动画加强，产生扫描线效果

## 技术方案
1. 保留原有背景图片和毛玻璃效果
2. 边框改为彩虹渐变动画（CSS动画旋转渐变角度）
3. 添加扫描线伪元素（::before），产生CRT扫描效果
4. 角落添加HUD装饰（L形角标）
5. 悬停时增强发光和动画速度

## 影响范围
- 仅修改 `/source/css/custom.css`
- 保留原有背景设置
- 不影响内容布局
