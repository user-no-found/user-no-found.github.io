# 变更提案：优化雪花特效自然度

## 概述
使用 Simplex Noise 生成流场，让雪花轨迹更自然、不再死板固定。

## 需求场景
### 场景 1：雪花飘落
- Given 用户打开页面
- When 雪花开始飘落
- Then 雪花轨迹自然随机，有漂浮感，不是固定的正弦曲线

### 场景 2：鼠标扰动
- Given 雪花正在飘落
- When 用户移动鼠标
- Then 附近雪花受气流影响产生自然偏移

## 技术方案
1. 实现轻量级 Simplex Noise 算法（内嵌，不引入外部库）
2. 创建2D流场，基于噪声生成风向
3. 每个雪花根据位置采样流场获得偏移力
4. 添加鼠标位置产生局部气流扰动
5. 保留原有猫咪碰撞检测逻辑

## 影响范围
- 仅修改 `/source/js/effects.js`
- 不影响其他功能
